# Contributing to PromptPersonality / Contribuire a PromptPersonality

## 🇬🇧 English

Thanks for your interest in contributing! Here's how to add a new personality:

### Steps

1. **Fork** this repository
2. Choose the language folder (`en/` or `it/`)
3. Copy the template from `templates/personality-template.md`
4. Fill it in with your personality
5. Save your file in the `personalities/` folder with a descriptive name (e.g., `zen-master.md`)
6. Open a **Pull Request** with a clear title and description

### Guidelines

- Each personality must follow the template structure
- Write clear, detailed system prompts — the more context, the better the AI will perform
- Include at least 2 example interactions
- Keep it respectful — no offensive, harmful, or discriminatory content
- Test your prompt with at least one AI model before submitting
- If you can, submit both an Italian and English version

### Naming Convention

Use lowercase with hyphens: `creative-writer.md`, `life-coach.md`, `tech-mentor.md`

## 🇮🇹 Italiano

Grazie per il tuo interesse nel contribuire! Ecco come aggiungere una nuova personalità:

### Passaggi

1. Fai **Fork** di questa repository
2. Scegli la cartella della lingua (`it/` o `en/`)
3. Copia il template da `templates/personality-template.md`
4. Compilalo con la tua personalità
5. Salva il file nella cartella `personalities/` con un nome descrittivo (es. `maestro-zen.md`)
6. Apri una **Pull Request** con un titolo e una descrizione chiari

### Linee guida

- Ogni personalità deve seguire la struttura del template
- Scrivi system prompt chiari e dettagliati — più contesto dai, meglio l'AI interpreterà il personaggio
- Includi almeno 2 esempi di interazione
- Mantieni il rispetto — nessun contenuto offensivo, dannoso o discriminatorio
- Testa il tuo prompt con almeno un modello AI prima di inviarlo
- Se puoi, invia sia la versione italiana che inglese

### Convenzione nomi

Usa minuscolo con trattini: `scrittore-creativo.md`, `coach-di-vita.md`, `mentore-tech.md`
