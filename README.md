# 🎭 PromptPersonality

### An open-source collection of AI personality prompts — with a studio to talk to them.
### Una raccolta open-source di prompt per dare personalità alle AI — con uno studio per parlare con loro.

---

## 🇬🇧 English

**PromptPersonality** is an open-source project where we design, refine, and share system prompts that give AI models a unique personality. But we don't stop at writing prompts.

### What's inside

- **Personality Prompts** — Ready-to-use system prompts that turn any AI into a distinct character: a mentor, a storyteller, a strategist, a creative partner, and more.
- **Templates** — A standard format so anyone can contribute new personalities.

### How to use

1. Browse the [`en/personalities/`](en/personalities/) folder
2. Pick a personality that interests you
3. Copy the system prompt into your favorite AI tool (ChatGPT, Claude, etc.)

### Contribute

We welcome contributions! Check out [CONTRIBUTING.md](CONTRIBUTING.md) to learn how to submit your own personality prompt.

## 🇮🇹 Italiano

**PromptPersonality** è un progetto open-source dove progettiamo, affiniamo e condividiamo system prompt che danno alle AI una personalità unica. Ma non ci limitiamo a scrivere prompt.

### Cosa trovi

- **Prompt Personalità** — System prompt pronti all'uso che trasformano qualsiasi AI in un personaggio distinto: un mentore, un narratore, uno stratega, un partner creativo, e molto altro.
- **Template** — Un formato standard per permettere a chiunque di contribuire con nuove personalità.

### Come usarlo

1. Esplora la cartella [`it/personalities/`](it/personalities/)
2. Scegli una personalità che ti interessa
3. Copia il system prompt nel tuo strumento AI preferito (ChatGPT, Claude, ecc.)

### Contribuisci

I contributi sono benvenuti! Leggi [CONTRIBUTING.md](CONTRIBUTING.md) per scoprire come inviare la tua personalità.
